Driver Records Dataset
Guyana Driver Record Management System Datasets
Overview

This repository contains sample datasets designed for use in a Driver Record Management System simulation. The datasets model real-world entities such as drivers, vehicle registrations, and accident records within the Guyanese context.

Datasets Included
Driver.csv – Contains demographic and identification details of drivers
Motor_Vehicle_Registration.csv – Contains vehicle registration and ownership records
Accident.csv – Contains accident reports and related details
Data Source & Generation

These datasets were synthetically generated using ChatGPT (OpenAI GPT-5.3).
The data was structured to reflect realistic formats based on Guyanese systems, including:

Address conventions (e.g., lot numbers and street names)
Vehicle plate formats (e.g., HC/HD for hire cars)
Law enforcement and regional references
Purpose

The datasets are intended for:

Database design and testing
Simulation of traffic and licensing systems
Academic and educational use
